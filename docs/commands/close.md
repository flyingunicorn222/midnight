# /close

Close one or multiple positions in a market.

``` bash
/close eurusd
```

Close all your EUR/USD Positions

``` bash
/close 1131761
```

Close your Position #1131761

#### Notes

1. For your convenience this command can also be triggered by issuing the
`/c` command
